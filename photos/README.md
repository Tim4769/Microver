# Photos

Project photos for Microver design, build, testing, and competition documentation.

## Folders
- `2024-2025/` - Earlier design, testbench, and development photos.
- `2025-2026/` - Final rover, build, electronics, and competition-season photos.

Keep board-specific engineering photos inside the relevant board folder when they document a PCB revision. Use this folder for project-level photos.
