# PCB Testing Firmware 2026

Final-stack PCB testing and electronics bring-up material for the Microver 2026 hardware.

## Folders
- `controllers/` - Bring-up sketches for the final compute/control hardware.
- `docs/` - Handoff documentation for software, power, and actuator control.

Start with `docs/board-validation-software-handoff.md` for board validation pin maps and `docs/power-and-actuator-control-handoff.md` for power/actuator behavior.
