# PCB Testing Firmware 2026 Controllers

Arduino Nano Every bring-up sketches for the 2026 final compute and control boards.

## Files
- `simple_motor_test/simple_motor_test.ino` - Minimal single-driver motor test.
- `minimum_one_wheel_test/minimum_one_wheel_test.ino` - Minimal one-wheel bring-up test.
- `one_side_spin_test/one_side_spin_test.ino` - One-side drivetrain spin test.
- `both_sides_drive_test/both_sides_drive_test.ino` - Both-sides drive test using the final control-board pin map.

Lift the rover or otherwise unload the drivetrain before running motor tests for the first time.
