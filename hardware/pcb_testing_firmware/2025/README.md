# PCB Testing Firmware 2025

Arduino Nano Every sketches, diagnostics, docs, and references used for PCB testing and electronics bring-up.

## Folders
- `controllers/` - Main integrated Arduino sketches.
- `diagnostics/` - Subsystem test sketches for drivetrain, serial link, and servos.
- `docs/` - Firmware-facing notes.
- `references/` - Datasheets and component notes used during bring-up.

For newer PCB testing notes, see `../2026/README.md`.
